> 全国大学生计算机设计大赛 · 人工智能应用赛道

---

## 📐 总体设计原则

1. **总时长**：15 页正式内容 + 2 页收尾 = 17 页，控制在 8~10 分钟
2. **开场要狠**：前 30 秒抓住评委注意力（数据冲击 + 真实故事）
3. **中段要稳**：技术展示不超过 4 页，留给 Demo 更多时间
4. **Demo 占比**：10~20% 的答辩时间应该是真实系统演示（建议 F9 启动 Demo Mode 或录屏播放）
5. **色调**：深色科技风（参考大屏配色）+ 绿色点缀（青苗 + 希望）
6. **字体**：标题用思源黑体 Heavy / Bold，正文用思源黑体 Regular，不用楷体宋体
7. **每页只讲一个点**：不要密密麻麻堆文字，每页一句话能概括
8. **数据优于文字，图表优于数据，动画优于图表**

## 🎯 制作建议

### 工具
- **PPT**：Keynote（Mac）/ PowerPoint 2021+ / WPS 2023，**不要用传统宋体模板**
- **模板参考**：搜索"AI 大屏 深色科技风 PPT 模板"或用 [office plus](https://www.officeplus.cn/)，找"大数据"主题
- **配色**：
  - 主色 `#22c55e`（青苗绿）`#0ea5e9`（科技蓝）
  - 辅色 `#a855f7`（AI 紫）`#ef4444`（预警红）
  - 底色 `#020617` / `#0b1224`（深色背景）
  - 文字 `#f1f5f9` / `#cbd5e1`
- **字体**：标题 **思源黑体 Heavy**，正文 **思源黑体 Regular**
- **图标**：[Tabler Icons](https://tabler.io/icons) 或 [Element Plus Icons](https://element-plus.org/zh-CN/component/icon.html) 截图

### 动画
- 使用 PowerPoint 的「平滑切换」（Morph）让页面之间过渡自然
- 数字统计页用「数字跳跃」动画
- 不要用粗俗的「飞入飞出」动画

### 讲稿背诵
- 每页讲稿字数控制在 100 字左右（讲完约 20~30 秒）
- 总讲稿 3~4 张 A4 纸
- 熟练度：背 3 遍 + 模拟演讲 3 遍

### 排练
- **3 天前**：PPT 定稿
- **2 天前**：完整排练，计时
- **1 天前**：向指导老师汇报一遍，收集建议
- **当天**：开演前喝温水、深呼吸、不要吃太饱

**时长**：**90 秒（推荐）** 或 120 秒（最多）

**制作工具**：
- 录屏：OBS Studio（免费）
- 剪辑：剪映 / Premiere / DaVinci Resolve
- 配音：剪映自带 AI 配音（选"尹甜甜"或"解说小帅"）
- 配乐：选用《Time》（Hans Zimmer）或《Corynorhinus》（Hans Zimmer）开头段

**脚本**：详见 `docs/演示视频脚本.md`

**拍摄技巧**：
- 录屏前确保 `/ethics`、`/enhance/*`、`/dashboard` 等所有新页面都能跑通
- **提前运行** `python scripts/seed_demo_extras.py` 让所有数据丰富
- 浏览器字体缩放到 110%，方便评委看清
- 演示时先启动 Demo Mode（F9），录 15~30 秒自动播放镜头
