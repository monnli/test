## 一、项目背景与社会价值（1 页）

- 中国 17.5% 青少年存在心理健康问题（《中国国民心理健康发展报告》）
- 学校心理老师配比严重不足（教育部数据：1:2000 ~ 1:5000）
- 传统心理筛查方式：单点、低频、滞后、依赖主观报告
- AI 多模态分析的潜力与必要性

## 二、行业痛点与现有方案不足（1 页）

- 痛点 1：单一维度（仅量表 / 仅成绩）→ 我们做多模态融合
- 痛点 2：黑盒预警 → 我们做可解释归因
- 痛点 3：监控感强 → 我们做"辅助而非替代"伦理设计
- 痛点 4：技术堆砌、不落地 → 我们做完整 SaaS 平台

## 四、核心算法详解（3 页）

### 4.1 多模态融合
- 图像：YOLOv8 行为检测、InsightFace 人脸识别、HSEmotion 表情
- 文本：通义千问 + 关键词词典双保险
- 量表：5 套国际/国内标准量表（PHQ-9/GAD-7/SCARED/CES-DC/MHT）

### 4.2 关联分析
- 数据矩阵：心理指数 / 学业 / 量表 / 课堂行为 / 综合风险
- Pearson 相关系数热力图
- 散点矩阵识别异常群体

### 4.3 风险评分公式
- 加权和：心理 35% + 量表 20% + 文本 15% + 对话 15% + 学业 10% + 课堂 5%
- 4 级阈值：green < 20 < yellow < 40 < orange < 60 < red

> **全文口径**：大屏、视频报告、心理量表、预测、聚类、干预、伦理仪表盘等**所有可量化指标**的逐条定义与两套「参与度」区别，见仓库根目录 **`算法.md`**（与代码同步维护）。

### 4.4 自动归因
- 每条预警附带原因清单，可解释 AI

## 五、关键技术亮点（2 页）

1. **流水线基类 + Mock 降级**：任何机器都能跑通，演示机零依赖
2. **本地化 AI**：可全部部署在校园内网，敏感数据不出校门
3. **WebSocket 实时**：摄像头帧实时分析 + Canvas 叠加
4. **异步任务零依赖**：Python threading，无需 Celery / Redis
5. **PDF 报告自动生成**：reportlab + 中文字体 + AI 综合建议
