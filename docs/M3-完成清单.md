# M3 · 课堂视频分析完成清单

## ✅ 已完成

### 数据模型
- 6 张新表：class_sessions / videos / analysis_tasks / behavior_records / emotion_records / attendance_records

### 后端
- 视频上传 + MD5 去重 + OpenCV 视频元信息探测（无 OpenCV 时退回默认值）
- 异步分析 worker：基于 Python threading（无需 Celery / Redis）
- 智能抽帧：装了 OpenCV 真实抽帧，否则按时间间隔生成 mock 帧，**无论是否真有视频流都能完成分析**
- 分析报告：行为时间轴 / 情绪饼图 / 情绪曲线 / 4 项指标
- WebSocket 实时摄像头：`/ws` 命名空间，前端发 frame，后端调 AI 服务回传结果
- 分析任务进度通过 DB 持久化 + WebSocket 实时推送

### REST 接口
- `GET  /api/classroom/videos`
- `POST /api/classroom/videos`（multipart 上传）
- `POST /api/classroom/videos/{id}/analyze`
- `GET  /api/classroom/videos/{id}/tasks`
- `GET  /api/classroom/tasks/{id}`
- `GET  /api/classroom/tasks/{id}/report`

### 前端 5 个页面
- 视频库（列表 + 上传入口）
- 上传视频（带班级选择 + 抽帧间隔配置）
- 视频任务记录
- 任务详情（进度条 + 4 指标 + 行为时间轴 + 情绪饼图 + 情绪曲线 + 视频回放）
- 实时摄像头（WebRTC 取流 + Canvas 叠加检测框 + Socket.IO 推送）

### 测试
- 24 个后端测试全部通过（新增 3 个课堂分析测试，覆盖上传/分析/报告）
