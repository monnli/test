# 青苗守护者 · 5 分钟演示视频脚本

## 制作建议

- **录屏**：用 OBS Studio，1920×1080，30fps
- **配音**：女声沉稳清晰，语速适中
- **配乐**：开场用《Time》（Hans Zimmer），转场用轻柔钢琴
- **片头/片尾**：3 秒静态卡片 + 项目 logo

## 镜头脚本

### 片头（0:00~0:15）

**画面**：黑底→渐显 logo + 文字「青苗守护者」+ 副标题「AI 中小学生心理健康综合分析平台」

**配音**：「中国 17.5% 的青少年存在不同程度的心理健康问题。每一个数字背后，都是一个家庭。」

### 第一幕：登录与角色切换（0:15~0:45）

**画面**：
1. 浏览器打开登录页（深色科技背景 + 演示账号一键填充按钮）
2. 点击「学校管理员」→ 跳转工作台（彩色卡片 + 折线图）
3. 退出 → 切换「班主任 head_7_1b」→ 数据范围明显缩小
4. 退出 → 切换「物理科任老师」→ 又是不同范围

**配音**：「平台支持 5 级精细权限：学校管理员、心理老师、年级组长、班主任、科任老师，每个角色看到的数据范围根据其职责自动限定。」

### 第二幕：课堂视频分析（0:45~1:45）

**画面**：
1. 进入「课堂分析 → **上传视频**」（或先到「**视频库**」看列表），选一段约 10 秒课堂视频（提前录好）
2. 选择班级、抽帧间隔 → 上传
3. 跳转**任务详情**页：进度条到 100% 后查看报告
4. 指标卡 + 行为时间轴（**标准八类**堆叠）+ 情绪分布 + 曲线
5. 打开「**视频回放**」播放原片；暂停后展示**当前帧 YOLO 检测画框**（单帧可解释）

**配音**：「上传课堂视频后，系统按间隔抽帧，调用 YOLOv8 行为检测与表情识别，生成结构化报告：八类课堂行为与情绪随时间变化一目了然；回放时还可对单帧实时画框，辅助讲解。」

### 第三幕：实时摄像头（1:45~2:15）

**画面**：
1. 进入「实时摄像头」，授权摄像头
2. 镜头对准团队 3 人，画面上实时叠加绿色检测框 + 表情标签
3. 一人举手，框立刻识别

**配音**：「我们也支持实时分析。WebRTC 推流到后端，毫秒级返回结果。教师可以在直播课中即时了解全班状态。」

### 第四幕：心理量表（2:15~2:45）

**画面**：
1. 进入「心理健康」→ 显示 5 套量表（PHQ-9 / GAD-7 / SCARED / CES-DC / MHT）
2. 点击「PHQ-9 下发测评」→ 选学生 → 点选 9 道题答案
3. 提交 → 弹出评级结果（轻度抑郁，建议关注）

**配音**：「集成 5 套国际国内标准量表，自动评分、自动分级。结果直接进入学生档案，触发预警时同步通知心理老师。」

### 第五幕：学生心理档案（2:45~3:15）

**画面**：
1. 跳转学生档案页
2. 综合心理指数 88（绿色）+ 风险等级
3. 60 天情绪曲线图
4. 文本分析记录、AI 对话历史

**配音**：「每位学生都有完整的心理健康档案：60 天情绪轨迹一目了然，文本分析、量表测评、AI 对话全部沉淀，老师可以追溯任何时间点。」

### 第六幕：AI 心理对话（3:15~3:45）

**画面**：
1. 在档案页点「开启新对话」
2. 输入「我最近压力很大」→ AI 温和回复
3. 输入「我想去死」→ AI 立即返回「请联系老师或拨打 12320」+ 系统右上角弹出「高风险」警告

**配音**：「AI 心理辅导员不能替代医生，但可以成为学生 24 小时在线的倾诉对象。一旦检测到自伤风险，系统立即标记并通知心理老师，绝不延误。」

### 第七幕：关联分析与预警（3:45~4:30）

**画面**：
1. 进入「关联分析」→ 散点图（心理 vs 学业）+ Pearson 热力图
2. 进入「预警中心」→ 4 级统计卡 + 列表
3. 点击一条 red 预警 → 显示原因清单（心理指数低、量表评级中度、文本检测到风险词）
4. 点击「干预记录」→ 添加一条「已与学生家长沟通」

**配音**：「这是我们的核心差异化——多维关联分析。综合 5 类数据交叉计算风险评分，每条预警都附带原因清单，做到可解释 AI。预警工单可流转、可追溯、可审计。」

### 第八幕：数据大屏（4:30~5:00）

**画面**：
1. 进入数据大屏（全屏沉浸式）
2. 88px 巨型心理指数旋转光晕
3. 6 张图表 + 实时预警流滚动；其中**今日课堂行为分布**与视频分析共用**八类**口径
4. 整屏视觉冲击，自动循环

**配音**：「最后是我们的数据大屏：全校心理态势、今日行为八类分布、预警流一屏收束。我们用一双 AI 慧眼，守护每一株青苗的成长状态。谢谢观看。」

### 片尾（5:00~5:10）

**画面**：黑底 + 团队成员名单 + Logo + 二维码

**配音**：（配乐渐弱）「青苗守护者 · 团队 [团队名]」
