# 青苗守护者 · 版本迭代总览（M0~M10 及近期收尾）

> 本页用于把多轮迭代后的系统能力“拉齐口径”：每个里程碑做了什么、在系统哪里能看到、答辩怎么演示。

---

## 0. 快速导航（答辩最常用的 6 个入口）

- **数据大屏**：`/dashboard`（一句话讲价值 + 全校态势）
- **工作台**：`/workbench`（老师日常入口）
- **课堂分析**
  - 视频库：`/classroom/videos`（含上传、删除、进任务详情）
  - 上传视频：`/classroom/upload`
  - 摄像头墙：`/classroom/cameras`
  - 实时直播：`/classroom/live/:cameraId`
  - 课表管理：`/classroom/schedule`
- **心理健康**：`/psychology`（量表/档案）
- **预警中心**：`/alerts`（四级风险 + 工单流转）
- **报告中心**：`/reports`（在线查看 + PDF 下载）

## 1. 权限与数据范围（所有模块共同遵循）

系统采用 5 级角色 + 数据范围授权，所有查询都基于（学校、年级、班级、科目）做过滤。

- **学校管理员 / 心理学老师**：本校全部
- **年级组长**：本年级
- **班主任**：本班全科 + 自任教科目跨班（班级×科目）
- **科任老师**：仅自任教的（班级×科目）
- **超级管理员**：平台全局（隐藏角色）

补充：

- **学校管理（新建/编辑/删除学校）仅超级管理员可用**；学校管理员账号默认只管理本校数据。

## 4. 部署与演示前检查（最常见踩坑）

- **演示数据**：先跑 `scripts/init_demo.bat` / `scripts/init_demo.sh`
- **一键启动**：用 `scripts/start_all.bat` / `scripts/start_all.sh`
- **Mock 模式**：演示机建议 `.env` 里 `FORCE_MOCK=true`（启动快、最稳）
- **权限入口**：
  - 学校管理员只能看本校
  - 学校管理仅超管（默认演示账号隐藏该入口）

## 6. 近期收尾迭代（M10 之后 · 与当前代码对齐）

> 以下能力已合入主线，答辩/演示口径以当前系统为准；细节公式见仓库根目录 **`算法.md`**。

- **课堂行为标准八类**：`BehaviorRecord` 历史标签经 `_normalize_behavior_cn` 归一到八类中文；**数据大屏「今日行为分布」**与 **视频任务报告**、**演示种子数据**（`scripts/seed_demo_extras.py`）口径一致。
- **视频库**：侧栏 **课堂分析 → 视频库** 列表；**上传视频** 独立入口；支持在权限范围内 **删除视频**（级联清理任务与行为/情绪记录）。
- **任务分析报告页**：**视频回放**；当前帧可调用 **`POST /classroom/videos/<id>/detect-frame`** 走 YOLO 实时画框（与异步整段分析任务独立）；静态资源经 **`/storage`** 代理与正确 MIME。
- **演示模式**：`GET /api/enhance/demo-script` 步骤与口播扩充；**F9** 开/关、**Esc** 退出、**空格** 暂停/继续；长页面在单步停留时间内 **自动纵向滚动**（主内容区或文档根），便于录屏看全页。
- **算法文档**：根目录 **`算法.md`** 汇总大屏、报告、心理、预警、预测、聚类、干预、伦理等指标与两套「参与度」区别。
