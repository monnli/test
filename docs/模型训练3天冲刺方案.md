## 📅 Day 1（今天）· 数据准备 + 开训

### 上午（2 小时）

#### Step 1. 装训练环境（20 分钟）
```bash
conda activate kt_env
pip install ultralytics

# 验证 GPU 可用
python -c "import torch; print('CUDA:', torch.cuda.is_available(), torch.cuda.get_device_name(0))"
# 应输出：CUDA: True  Tesla P100-PCIE-16GB
```

#### Step 2. 准备数据集（30~60 分钟）
```bash
python scripts/train/prepare_scb_dataset.py
```

流程：
- 自动从 GitHub clone SCB-dataset 仓库（约 500MB，可能需要梯子或耐心等待）
- 自动划分 train / val（9:1）
- 自动生成 data/scb/scb.yaml 配置

**如果 clone 很慢**：
1. 手动访问 `https://github.com/Whiffe/SCB-dataset`
2. 下载 ZIP → 解压到 `data/.cache/SCB-dataset/`
3. 再重新跑 `prepare_scb_dataset.py`

成功后终端会输出：
```
[OK] 训练集：3600 张  验证集：400 张
[OK] 配置写入 data/scb/scb.yaml
```

#### Step 3. 冒烟测试（10 分钟）
跑 5 epoch 确认流程 OK（真实训练之前的"体检"）：
```bash
python scripts/train/train_yolov8_behavior.py --epochs 5 --name smoke_test
```

看到 `✅ 训练完成` + 有权重文件生成即可。

### 下午~晚上（正式训练，约 8 小时）

```bash
python scripts/train/train_yolov8_behavior.py --epochs 100 --name scb_v1
```

**挂着跑就行**，P100 训练约 6~8 小时。

- 中途想看进度：打开 `runs/qingmiao_behavior/scb_v1/` 有 `results.png`
- 想中断：Ctrl+C。恢复：加 `--resume`
- 想用更大模型（更准但慢）：`--base yolov8m.pt --batch 8`

## 📅 Day 2（明天）· 评估 + 集成 + 测试

### 上午（2 小时）

#### Step 4. 评估模型（30 分钟）
```bash
python scripts/train/evaluate_model.py \
    --weights runs/qingmiao_behavior/scb_v1/weights/best.pt \
    --compare \
    --video storage/demo_videos/class_7_1.mp4
```

输出关键数字：
- `mAP50      ：0.XX`  ← **这是答辩要念的数字**
- `mAP50-95   ：0.XX`
- 每类 AP50（可以挑一两个说）
- 推理速度（ms/帧 和 FPS）
- 与官方 yolov8n 对比提升（证明训练的价值）

**把终端截图存下来，PPT 要用。**

#### Step 5. 集成到系统（15 分钟）
```bash

    --weights runs/qingmiao_behavior/scb_v1/weights/best.pt
```

这一步做了：
- 把 `best.pt` 复制到 `ai_service/models/yolov8_classroom.pt`
- 写入中文标签映射

## 🎤 答辩话术（重要）

### 介绍训练的页面

> "我们不是简单地使用开源模型，而是针对**中小学课堂**这个具体场景做了深度优化。
>
> 我们采集了 SCB 公开的课堂行为数据集，在 NVIDIA P100 GPU 上对 YOLOv8 进行了 100 轮微调。基础版 YOLOv8 只能识别通用的"人"和"手机"，我们训练的模型能识别 **举手、阅读、写作、站立、玩手机** 5~7 类中小学生典型课堂行为。
>
> 在验证集上，自训模型的 mAP50 达到 **0.XX**，比官方通用模型提升了 XX 个百分点，**更适合课堂场景**。"

### 如果评委问"训练数据怎么来的"

> "我们使用的是 GitHub 公开的 SCB 课堂行为数据集，这是北京师范大学等高校开源的中小学课堂标注数据集，约 4000~7000 张标注图像。使用合规、有出处。"

### 如果评委问"自训的精度够吗"

> "单一模型的精度不是我们的核心——这就是为什么我们做**多模态融合**。比如一个学生被遮挡，YOLO 识别置信度低，但 MediaPipe Pose 关键点和 InsightFace 人脸还能提供信息。多模态融合后综合判断置信度在实际场景中会显著提升，且每条结论都可解释。"

### 如果评委问"为什么没有训练人脸 / 表情"

> "人脸识别的 InsightFace buffalo_l 在 LFW 数据集上已达 99.83% 的 SOTA 精度，自行训练需要百万级人脸数据和 8 卡 A100 训 7 天，性价比不高。表情识别同样是成熟领域，HSEmotion 已是业内 SOTA。**我们把算力集中在课堂行为这个最独特、最有价值的场景上**。"

### 如果评委追问"还有哪些模型训练计划"

> "下一步我们会：
> - 在 CPsyCoun 心理咨询数据集上对通义千问 7B 做 LoRA 微调，进一步提升 AI 对话的专业性
> - 扩充课堂行为数据集，加入更多二级行为（如"抬头看老师"、"侧身说话"）
> - 用教师反馈数据做半监督学习，持续迭代。"

这几句是"储备答案"，如果评委没问可以不说。

## 💡 可能遇到的坑 & 解决方案

### 坑 1：`ultralytics` 装不上
```bash
pip install ultralytics -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 坑 2：训练时 CUDA OOM
```bash
# 减小 batch 或图像尺寸
python scripts/train/train_yolov8_behavior.py --batch 8 --imgsz 512
```

### 坑 3：SCB 数据集 clone 超时
手动下载 ZIP：
1. 浏览器打开 `https://github.com/Whiffe/SCB-dataset`
2. 点绿色 Code → Download ZIP
3. 解压到 `data/.cache/SCB-dataset/`
4. 重跑 `prepare_scb_dataset.py`

### 坑 4：训练到一半断电/Ctrl+C
```bash
# 直接 resume
python scripts/train/train_yolov8_behavior.py --name scb_v1 --resume
```

### 坑 5：自训模型加载但结果不对
检查：
```bash
cat ai_service/models/yolov8_classroom_labels.json
```
如果类别不对，手动编辑这个 JSON 即可。
