## 一条命令激活（推荐）

### A. 只接入你训练好的 `best.pt`（最贴近你当前需求）

```bash
python scripts/register_root_best_pt.py
```

### B. 一键接入更多真实模型（可选）

```bash
python scripts/activate_real_models.py --yolo-weights best.pt --api-key sk-xxxx
```

说明：

1. 会把 `best.pt` 部署到 `ai_service/models/yolov8_classroom.pt`
2. 从模型文件自动读出类别名，生成 `ai_service/models/yolov8_classroom_labels.json`（英文 → 中文映射）
3. 只有当你传入 `--api-key` 时，才会写入 `.env` 的 `DASHSCOPE_API_KEY`
4. 检查 mediapipe / insightface / hsemotion / dashscope 等依赖是否可导入
5. 打印每个流水线的激活建议

> 重要：上传视频分析要走真实 YOLO，请确保 `.env` 中 `FORCE_MOCK=false`，并安装 `ai_service/requirements.txt`（已包含 `ultralytics/torch/opencv`）。

## 常见问题

### Q: YOLOv8 加载成功但识别不到课堂行为
检查 `ai_service/models/yolov8_classroom_labels.json`。如果类别中文映射不准，手动编辑 `labels_cn` 字段，重启 AI 服务。

### Q: InsightFace 首次加载很慢
`buffalo_l` 是 280MB，首次会从 GitHub 自动下载。你可以预下载：
```bash
python -c "from insightface.app import FaceAnalysis; FaceAnalysis(name='buffalo_l').prepare(ctx_id=0)"
```

### Q: MediaPipe 报 libGL.so.1 not found
```bash
# Ubuntu
apt-get install libgl1 libglib2.0-0
# Windows 一般不会有这个问题
```

### Q: 通义千问返回 401
检查 `.env` 中 `DASHSCOPE_API_KEY` 没有多余空格或引号。

### Q: 我想临时回到 mock 模式
在 `.env` 设置 `FORCE_MOCK=true`（最推荐），或删除 `ai_service/models/yolov8_classroom.pt`。
